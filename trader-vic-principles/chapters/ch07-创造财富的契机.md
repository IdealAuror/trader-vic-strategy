# Chapter 7: 创造财富的契机：辨识趋势的改变

## Core Idea
尽早察觉趋势改变是在金融市场中最迅速而安全的赚钱方法——本章提供三套简单一致的技术方法（123准则、2B准则、ABC三波浪），用以辨识趋势变动并在趋势反转前后建立头寸，掌握中期或长期走势60%~80%的幅度。

## Frameworks Introduced
- **趋势线正确绘制法**：趋势线必须精确反映趋势定义——上升趋势线以最低低点为起点，向右上方连接最高高点前的某一低点，使该直线在两低点间不穿越任何价位；下降趋势线以最高高点为起点，向右下方连接最低低点前的某一高点，中间不穿越价位。一致性是核心，不可将个人欲望强加于趋势线之上。
  - When to use: 每次判断趋势变动之前，作为123准则的前提基础
  - How: （1）选定分析期间（长期数月数年、中期数周数月、短期数天数周）；（2）上升趋势：以期间内最低低点为起点，连接最高高点出现前的某个低点，确保直线在两低点间不穿越任何价位；（3）下降趋势：以期间内最高高点为起点，连接最低低点出现前的某个高点，确保直线在两高点间不穿越任何价位；（4）将直线延伸经过最高高点（上升）或最低低点（下降），观察价格是否穿越

- **123准则**：趋势变动的三道确认条件——①趋势线被价格穿越（突破）；②上升趋势不再创新高或下降趋势不再创新低（测试前高/前低失败）；③价格向下跌破前一波回档低点（上升趋势）或向上穿越前一波反弹高点（下降趋势）。三项全满足 = 趋势已变。
  - When to use: 任何市场（股票、期货、债券、外汇）的任何时间结构下，作为主要的趋势变动判定工具
  - How: （1）绘制正确的趋势线；（2）价格穿越趋势线 → 在穿越处标①；（3）画水平线穿越目前最高价（上升趋势），另画一条水平线穿越前一波回档低点；或画水平线穿越目前最低价（下降趋势），另画一条水平线穿越前一波反弹高点；（4）价格回升接近/触及当前高价的水平线但未能突破 → 标②；（5）价格跌破前回档低点水平线（上升）或突破前反弹高点水平线（下降） → 标③；（6）出现两种情况 → 趋势很可能变化；三种情况全出现 → 趋势已变，朝新方向发展；（7）在流动性高、对消息不敏感的市场中可靠性最高

- **2B准则**：在上升趋势中，价格创新高后未能持续上涨，随后又跌破先前高点；或在下降趋势中，价格创新低后未能持续下跌，随后又回升穿越先前低点——仅此单一现象即代表趋势极可能反转。其概率重要性高于123准则中任何单一条件。
  - When to use: 在123准则完全确认前提前建立头寸；适用于短期、中期、长期三种趋势；尤其适用于程序交易密集、流动性高的市场
  - How: （1）上升趋势转空2B：观察价格突破前高后无法站稳，快速回落跌破该前高价位 → 以止损单做空，止损设在假突破所创新高之上；（2）下降趋势转多2B：观察价格跌破前低后无法持续，快速回升突破该前低价位 → 以止损单做多，止损设在假突破所创新低之下；（3）短期2B：通常在一天内或更短完成；（4）中期2B：通常在3~5天内完成；（5）长期2B：通常在7~10天内完成；（6）出错立即认赔：价格再创新高（做空后）或再创新低（做多后）则立即平仓；（7）假突破时成交量偏低，反转确认时成交量急剧放大为最佳确认

- **ABC三波浪次级修正模式**：多头市场或空头市场的次级修正走势几乎都呈现ABC三波浪模式——这是斯波朗迪从艾略特波浪理论中唯一采纳的模式。BC浪成交量萎缩，C点是建立投机性头寸的理想位置。
  - When to use: 判断多头或空头市场中次级修正走势的底部和顶部，寻找沿主趋势方向重新入场的机会
  - How: （1）确认市场处于多头或空头主要趋势中；（2）识别修正走势呈现A-B-C三浪结构；（3）验证BC浪运行中成交量持续缩小；（4）当价格向上突破BC浪的下降趋势线时（多头市场次级修正），在C点附近建立投机性多头头寸；空头市场则反向操作；（5）止损点设在A点与C点中的较低价位（多头）或较高价位（空头）；（6）突破时若成交量无明显变化 → 底部形成可能性不大；若突破时放巨量 → 可能不是空头修正而是主要多头市场的第二只脚

## Key Concepts
- **趋势线**: 连接低点间（上升趋势）或高点间（下降趋势）且中间不穿越任何价位的直线，是判断趋势变动的基准工具；绘制缺乏一致性是业余与专业玩家最普遍的失误
- **123准则之①：趋势线突破**: 价格穿越经由正确方法绘制的趋势线，是趋势变动的第一个征兆
- **123准则之②：测试高低点失败**: 价格回升但未能突破前高（上升趋势），或反弹未能突破前低（下降趋势）——通常但非必然发生在趋势变动过程中
- **123准则之③：突破前回档低点/反弹高点**: 上升趋势中价格向下跌破前一个回档低点，或下降趋势中价格向上穿越前一个反弹高点——相当于道氏理论对趋势变动的确认
- **2B准则**: 价格创新高（低）后走势未能持续，随后反方向跌破前高（或穿越前低）；源自场内交易员的"清除止损"行为，是123准则的提前进场版本
- **ABC三波浪**: 斯波朗迪从艾略特波浪理论中唯一采纳的模式，描述次级修正的典型三波段结构，BC浪量缩、C点建仓
- **挨耳光（whipsaw）**: 在价格剧烈波动的市场中依据准则交易后行情突然反向反转，导致亏损出场
- **清除止损（taking out the stops）**: 场内交易员了解客户止损单集中设于前高或前低附近，故意将价格推至支撑稍下方或压力稍上方以触发止损单——这是2B模式发生的主要机制
- **流动性过滤**: 123/2B准则的可靠性取决于市场质量——高流动性、少跳空、对消息不敏感的市场成功率更高，反之频繁挨耳光
- **心理止损（mental stop）**: 预先设定但仅在价格接近时才递入交易指令的止损方式，与事前递入的正式止损单相对

## Mental Models
- **趋势变化"递进确认"思维**: 突破→测试失败→穿越确认，三步构成趋势变动的完整证据链。每多满足一项条件，趋势变动的概率递进增强；三项全满足则等于趋势已变。这不是"如果满足A则可能反转"的离散判断，而是概率递进的结构化思维
- **场内博弈/止损清除思维**: 2B模式不是偶然的价格噪声，而是专业交易员利用大众弱点的结构性结果——他们知道止损单集中在何处，驱动价格触发止损后反向操作。理解这一机制，才能将2B从"图形巧合"升级为有因果支撑的交易信号
- **流动性匹配思维**: 交易准则不是普适的——高流动性市场准则成功率远高于跳空频繁、成交量稀薄的市场。不是方法失效，是市场不匹配。选择市场本身就是交易决策的一部分
- **时间结构嵌套思维**: 2B准则在短中长三个时间维度同时可用，但完成时间不同（日内→3~5天→7~10天），且中长期2B成功率高于当日冲销的50%。投资者必须明确自己参与的时间级别，不可混用

## Anti-patterns
- **趋势线绘制缺乏一致性**: 若两低点（或两高点）连线之间穿越了任何价位，趋势线即不符合定义——基于错误趋势线的123判断等同于在错误前提上做决策，这是最普遍的致命失误
- **在流动性差或消息敏感市场机械套用123/2B**: 外汇、小盘股等市场经常跳空和剧烈反转，准则成功率大幅降低，导致频繁挨耳光；必须先过滤市场质量再套用准则
- **不设止损直接依据准则建仓**: 任何准则都不是100%有效——必须在进场前确认止损位（设定于先前支撑或压力价位），且该止损位技术上客观可执行
- **等待123三条件完全确认才进场**: 三项全满足时通常已错失一大段行情；应结合2B准则在第②步或2B成立时提前建仓，以较小的止损距离换取更大的盈利空间
- **将ABC修正的C点突破伴随巨量误判为买入信号**: BC浪突破时若成交量急剧放大，可能不是空头修正的结束而是主要趋势第二浪的启动——需配合基本面和其他指数确认

## Key Takeaways
1. 趋势线必须精确一致：上升线以最低低点连最高高点前的某低点，下降线以最高高点连最低低点前的某高点，两连接点之间绝不穿越价位；错误的趋势线=错误的判断基础
2. 123准则三步递进：①突破趋势线 ②测试前高/前低失败 ③跌破前回档低点/突破前反弹高点——三条件全满足=趋势已变；仅满足两项=趋势很可能变化，可轻仓试探
3. 2B准则是123准则的提前进场版本：创新高后跌破前高（或创新低后冲破前低）即趋势极可能反转；极小止损换极大盈利空间，盈亏比可达1:5以上，但胜率约50%，必须严格止损
4. ABC三波浪的C点是建仓最佳时机：BC浪量缩且价格突破BC趋势线时进场，止损设在A与C的极值价位；放量突破则需警惕，可能不是修正结束信号
5. 挨耳光的本质是市场不匹配：避开流动性差、跳空频繁、对消息敏感的市场，只在流动性充裕的市场中运用准则；出场点必须在进场前就已确认
6. 仅需少数简单工具即可高效筛选机会：1小时内浏览两三百幅图形，用123/2B/ABC从中筛选5~10个理想对象做进一步研究
