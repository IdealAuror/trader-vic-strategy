# Chapter 5: 趋势的定义

## Core Idea
趋势必须被精确定义才能判断其变化——上升趋势是高点与低点不断垫高的系列走势，下降趋势则相反；唯有明确界定趋势，才能一致性地识别买卖时机。

## Frameworks Introduced
- **趋势的量化定义**：上升趋势 = 连续涨势持续突破前高 + 回调不跌破前波低点；下降趋势 = 连续跌势持续跌破前低 + 反弹不突破前波高点。
  - When to use: 任何市场、任何时间结构下，判断当前趋势方向与趋势变化
  - How: （1）选择时间周期（长期/中期/短期）；（2）上升趋势观察高点是否持续创新高、低点是否持续垫高；（3）下降趋势观察低点是否持续创新低、高点是否持续下滑；（4）将定义应用于具体价格序列，不依赖主观感觉

- **指数相互验证原则**：所有相关指数必须相互确认，单一指数的走势结论几乎必然错误。
  - When to use: 判断大盘趋势方向，尤其是关键转折点
  - How: （1）同时观察多个主要指数（工业指数、运输指数、S&P 500等）；（2）只有当多个指数同时发出同一方向信号时才确认趋势；（3）若仅一种指数突破而其他未确认，视为背离——趋势可能生变

- **市场四阶段模型**：任何市场必处于承接、出货、趋势（上涨或下跌）或整理四种技术阶段之一。
  - When to use: 判断当前市场处于什么状态，确定交易策略
  - How: （1）窄幅盘整（2-3周以上，约5%波动区间）表示承接或出货；（2）价格向上突破盘整区间上限 = 承接完成，做多信号；（3）价格向下跌破区间下限 = 出货完成，做空信号；（4）盘整中若方向不明，等待突破确认再行动

- **价量关系法则**：超买行情中价涨量缩/价跌量增；超卖行情中价跌量缩/价涨量增。
  - When to use: 辅助验证趋势的健康程度和市场心理状态
  - How: （1）在趋势行情中观察成交量与价格的关系；（2）超买状态：价格创新高但成交量萎缩，表明动能衰竭；（3）超卖状态：价格创新低但成交量萎缩，表明卖压耗尽；（4）仅作为辅助工具，不作为主要决策依据

## Key Concepts
- **上升趋势**：由一系列连续涨势构成，每一段涨势持续向上突破先前高点，中间回调不跌破前一波跌势低点
- **下降趋势**：由一系列连续跌势构成，每一段跌势持续向下跌破先前低点，中间反弹不突破前一波涨势高点
- **背离**：一种指数出现反转走势，另一种指数呈现相反方向——仅有负面警示功能，提示趋势可能改变
- **窄幅盘整**：长达2-3周及以上的走势，两种指数大约在5%的价格区间内波动，可能代表承接或出货
- **承接**：长期投资者在底部区域分批买入，不推高价格，形成窄幅盘整后向上突破
- **出货**：长期投资者在顶部区域分批卖出，不明显压低价格，形成窄幅盘整后向下突破
- **整理**：趋势中途的获利了结调整，发生在已确认的趋势中
- **突破**：价格穿越盘整区间的上限或下限——对交易者、投机者、投资者均唯一重要的走势信号
- **超买/超卖**：超买由感觉与预期驱动，掌握信息的人已离场；超卖则股价偏低，精明投资者开始承接

## Mental Models
- **多时间框架思维**：同一张价格图中同时存在三种趋势（长期、中期、短期），回答"趋势如何"之前必须先明确所指的是哪个时间周期的趋势。不同周期趋势可能方向相反，交易者必须知道自己参与的是哪个级别。
- **验证性思考**：单一信号不可信，必须等待其他独立信号交叉确认。这是斯波朗迪整个交易哲学的核心——不赌单一判断，增加胜算。指数相互验证、价量配合验证都属于这一思维。
- **市场心理四阶段循环**：承接（聪明钱进场）→ 上升趋势（群众跟进）→ 出货（聪明钱离场）→ 下降趋势（群众恐慌），周而复始。识别当前阶段比预测方向更重要。

## Anti-patterns
- **仅根据单一指数判断趋势**：道氏理论明确要求两种指数相互验证，单一指数信号几乎必然错误。1987年崩盘后若只看一只指数将完全误判。
- **缺乏一致性标准地画趋势线**：大多数交易者随意绘制趋势线导致无效结论。突破一条画错的趋势线不代表任何意义。
- **混淆不同时间周期趋势**：在长期下跌趋势中看到中期反弹就认为趋势反转，或回答"趋势如何"时未指明时间周期。

## Key Takeaways
1. 趋势定义是客观的、可操作的：高点低点序列决定方向，不依赖主观感觉
2. 所有相关指数必须相互确认，背离仅具负面警示用途
3. 窄幅盘整后的突破是唯一对所有市场参与者都重要的信号
4. 价量关系反映市场心理——超买区价涨量缩意味着动能衰竭
5. 首先确定自己交易的是长期/中期/短期趋势，再应用相应定义
6. 每日波动单独来看几乎必然错误，需积累成可辨识的模式才有预测价值
