# Chapter 8: 失败可能来自于分析报告未提供的资料

## Core Idea
交易失败往往不是因为分析报告说错了什么，而是因为报告根本没有提及某些关键信息。斯波朗迪不再阅读任何分析报告，而是直接从市场本身提取所需信息。本章约80%的篇幅用于讲解三套辅助性技术工具——移动平均线、相对强度、震荡指标——以及如何用它们替代分析报告的功能。

## Frameworks Introduced

### 一、移动平均线交易准则 (Moving Average Trading Rules)

移动平均线的核心作用是平滑价格噪音、凸显趋势方向。它不预测未来，只陈述当前趋势状态。

**1. 200天期移动平均线（个股与指数）**
- 适用场景：确认道氏理论的长期趋势方向，筛选个股多空资格
- How：
  1. 均线由下降转为平坦/上升，且价格由下往上穿越均线 → 主力买进信号
  2. 均线由上升转为平坦/下降，且价格由上往下穿越均线 → 主力卖出信号
  3. 个股硬规则：价格低于200天均线绝不买进；价格高于200天均线绝不卖出
  4. 多头市场中的第一波次级修正走势，价格可能跌破200天均线——这是正常现象

**2. 10周/30周移动平均线交叉系统**
- 适用场景：中期趋势判定，比200天均线更灵敏
- How：
  1. 买进条件（三者缺一不可）：10周均线上穿30周均线；两条均线斜率均向上；价格位于两条均线之上
  2. 卖出条件（三者缺一不可）：10周均线下穿30周均线；两条均线斜率均向下；价格位于两条均线之下
  3. 强调三条件同时成立才执行——任何一条件不满足就等待

**3. 商品期货中的移动平均线**
- 不存在固定最优周期——需针对特定商品反复测试，寻找最适配期间
- 失效时及时调整周期参数——执著于某个曾经有效的周期是常见错误

**4. 核心纪律**
- 绝不执著于任何单一指标或周期
- 均线的滞后性是设计意图而非缺陷——平滑噪音必然牺牲灵敏度
- 移动平均线是"交通红灯"：它告诉你何时不能通行，而非何时可以通行

### 二、相对强度运用法则 (Relative Strength Application)

相对强度衡量个股/商品相对于大盘或同类资产的价格表现比率，用于识别领导股和弱势股。

- When to use：选股（做多/做空）、商品价差交易、确认市场领导股
- How：
  1. 做多选股：在上升趋势中，选择与大盘指数同日或更早创新高的股票——这些是市场领导股
  2. 但不是创新高时立即买进——等待回档，在跌势中入场，因强势股迅速回升的概率更高
  3. 做空选股技巧：不卖空最强的股票（万一判断错误损失惨重），不卖空最弱的股票（下行空间有限），选择相对强度适中的股票
  4. 卖空强势股的时机：工业指数与运输指数均已出现中长期顶部，且1-2-3准则确认后，强势股顶部可能出现1-3天恐慌性暴跌
  5. 商品市场：比较同类商品（黄金vs白银、玉米vs小麦）选择最强品种做多或构建价差头寸
  6. 硬约束：相对强度再强，若价格低于移动平均线，仍不考虑买进

### 三、震荡指标辅助确认 (Oscillator Confirmation)

震荡指标以行情宽度或价格为参数计算市场动能，在趋势转折点发出预警。它不单独作为交易依据，仅用于辅助确认。

- When to use：与道氏理论趋势变动信号配合，作为辅助确认工具
- How：
  1. 行情宽度震荡指标（短期10天/长期30天）：计算每日上涨与下跌家数之净值的移动总和，与道琼斯指数对比观察背离
  2. 价格震荡指标：计算5天价差的10天期净移动总和——数值越高/越低，中期趋势变动的可能性越大
  3. 商品震荡指标：以两个移动平均值的差值构成，需针对特定商品反复测试适配周期
  4. 使用原则：仅作为辅助工具——震荡指标告诉你"需要警惕"，但不告诉你会发生什么或何时发生
  5. 商品震荡指标的可靠性远不如股票指数，不可单独依赖

### 四、KISS原则 (Keep It Simple, Stupid!)

只采用少数几个经过时间验证的辅助工具，不可过度复杂化分析系统。工具的边际效用递减——每增加一个指标，分析瘫痪的风险增加一分。

## Key Concepts

- **秘鲁鳗鱼逻辑**：如果你根据特定市场的知识交易，则你很可能因一件你不知道或未曾考虑的事件而失败。市场的因果链远超个人认知半径，但你不需要知道全部链条——只需要知道如何从市场本身提取信号
- **辅助性工具的正确角色**：技术指标的核心功能是告诉你"不可以做什么"，而非"可以做什么"——它们是交通红灯，提醒谨慎，不给方向。方向由道氏理论确定
- **盈余增长变动率**：基本面中最具统计有效性的指标——价格趋势的斜率在时间上领先盈余趋势斜率约3个月，前提是两者在6个季度以上的期间内存在显著相关性
- **盈余增长的复利效应**：盈余增长率25%意味着3年内盈余翻倍——这才是衡量成长潜能的核心因素，远胜于静态的市盈率或股息率
- **市盈率与账面价值的正确用法**：不应直接用于选股（低市盈率背后必有市场已知的理由），而应比较整体市场与历史的市盈率水平，判断超买/超卖状态
- **股息收益率的陷阱**：高股息收益率可能是因为股价持续下跌而盈余衰退——不考虑盈余增长的收益率数据具有误导性
- **信用扩张程度**：选股时的基本面考量——多头选择信用扩张程度低的公司（抗银根紧缩），空头则相反
- **移动平均线的钝化效应**：均线周期越长，滞后性越大——这是设计用途而非缺陷，其目的正是平滑噪音、凸显趋势
- **指标执著**：对任何技术指标的执著是最严重的错误——市场永远在变化，任何不承认这一事实的方法已经犯了根本性错误
- **折中主义**：最成功的投机者既非纯粹技术派也非纯粹基本面派，而是结合两者——以技术分析为主、基本面统计资料为辅

## Mental Models

- **已知的未知 vs 未知的未知**：分析报告的真正危险不在于它说错了什么，而在于它完全没有提到的东西——你无法防范你看不到的东西。解决之道是停止依赖二手信息，直接从市场提取一手信号
- **指示灯模型**：将辅助性技术工具视为铁路平交道上的红绿灯——它们不会告诉你火车何时到达或从哪个方向来，但会提醒你是否需要减速或可以全速前进。方向判断仍由道氏理论（相当于铁路调度中心）负责
- **工具生命周期**：每一个技术工具都有有效期，市场结构变化时工具会失效——必须果断抛弃，绝不动感情

## Anti-patterns

- **执著于单一指标**：相信某指标是"终结所有准则的准则"——市场永远在变化，任何不变的准则终将失效
- **过度信息收集**：运用极复杂的震荡指标或试图穷尽所有知识——违反KISS原则，最终导致分析瘫痪
- **纯学派思维**：纯粹依赖技术分析或纯粹依赖基本面分析——两者都难以获得稳定业绩
- **用市盈率/账面价值直接选股**：格雷厄姆与多德方法（买低市盈率、低账面价值股票）忽视了"股价下跌必有理由"——低市盈率背后往往是市场已知的困境
- **把辅助工具当方向工具**：用震荡指标或相对强度单独决定交易方向，而未经道氏理论确认主要趋势——这是最危险的误用

## Key Takeaways

1. 你不可能知道所有知识——关键是知道你在寻找什么，让市场提供你所需的大部分信息
2. 技术工具是"交通灯"而非方向盘：提示危险，不指示方向——方向由道氏理论确定
3. 移动平均线两条铁律：价格低于200天均线绝不买进，高于200天均线绝不卖出
4. 10/30周均线交叉需三条件同时成立：交叉、斜率向上、价格在均线上方
5. 相对强度选股：选与指数同日或更早创新高的领导股，但等待回档入场
6. 震荡指标仅用于辅助确认趋势变动信号，不可单独作为交易依据
7. 盈余增长率是最被低估的基本面指标——25%增长率意味着3年翻倍
8. KISS原则：只用少数经过长期验证的指标，复杂系统的边际收益往往为负
9. 技术工具失效时果断抛弃——不要让感情涉及交易准则
