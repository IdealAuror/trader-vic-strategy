# Chapter 3: 一致性成功的事业经营哲学

## Core Idea
交易必须当作事业经营，所有决策遵循三层递进原则：先在每次交易中优先保障资本，再追求一致性的获利能力，最后才以已有利润积极承担风险博取卓越回报。

## Frameworks Introduced
- **一致性成功三原则（层次递进框架）**：资本保障 → 一致性获利 → 卓越回报
  - When to use: 任何市场决策之前，作为最高指导原则
  - How:
    1. 资本保障：每次决策先问"潜在亏损多大"，再问"潜在利润多大"；风险/回报比不足1:3时持有现金，不计较是否跑赢大盘
    2. 一致性获利：以辅助原则运作——将已实现获利的一部分存入银行、一部分用于扩大下一笔头寸；亏损时按比例缩小头寸，确保任何时候都不会亏光风险资本
    3. 卓越回报：仅当已有累积获利时，以获利的一部分（而非原始资本）积极承担更大风险，即使判断完全错误也只损失先前的半数获利

- **季度滚动仓位管理法**：
  - When to use: 以一个固定周期（如季度）为操作基础时
  - How:
    1. 期初新头寸不超过总风险资本的10%，止损设在潜在损失不超过总资本的1%~2%
    2. 第一笔交易亏损后，下一笔头寸按亏损金额等比例缩小
    3. 第一笔交易获利后，存入一半利润，另一半用于放大下一笔头寸
    4. 遵循1:3风险/回报比，即使三笔只对一笔，长期仍有可观回报

## Key Concepts
- **资本保障**：三原则中最核心的原则，风险是任何潜在市场活动中首要考虑因素
- **辅助原则**：直接派生于更根本原则的概念——一致性获利是资本保障的辅助原则，因为资本不是静态数量
- **绝对回报 vs 相对回报**：唯一的问题是"我有没有赚钱"，而非"我是否跑赢大盘"；账册上的数据是绝对的，2+2必然是4
- **1:3风险/回报比率**：最大可接受的比率，即潜在回报至少是潜在损失的3倍
- **孤注一掷**：将过多风险资本投入单一头寸，是最严重、最易触犯的错误
- **黑与白的绝对观**：钞票不是绿色，是黑色或白色，对应正确与错误；拒绝基金管理圈子推销的"灰色"相对主义
- **50%获利保护法则**：任何既有的获利都不可再损失50%，这是对"让获利头寸持续发展"的具体解释
- **风险/回报恶化时的正确行为**：不关心是否超越大盘，直接转为现金或国库券

## Mental Models
- **棒球打击率模型**：最佳选手的打击率也只有30%~40%，但安打效益总大于三振伤害——交易同理，回报必须始终大于风险，不需追求高胜率
- **企业经营模型**：像谨慎的企业家一样经营交易，首先确保能够维持日常开销（保障资本），再追求盈余的稳定成长（一致性获利），最后才用一部分盈余进行较高风险的业务扩张（卓越回报）
- **账册绝对性模型**：以会计账册的绝对逻辑对抗投资界的灰色妥协文化——下跌10%如果仍优于下跌20%的大盘，这并非成功，账册上亏损就是亏损
- **概率累进模型**：基于1:3风险/回报比，即使只有33%的胜率，长期仍能稳定获利；关键在于永远留有筹码（通过动态仓位缩放实现）

## Anti-patterns
- **孤注一掷**：事前没有设定经营哲学的框架，将过多风险资本投入单一头寸，这是破产的最常见原因
- **在多头市场顶部追逐兼并股/LBO**：顶部附近LBO的风险/回报关系极不合理，唯一可接受的方法是期权小量参与且提前离场
- **满足于相对回报**：接受"跌得比大盘少就是胜利"的灰色逻辑，忽视账册上的绝对亏损数字
- **不持有现金**：当市场风险/回报比恶化时仍留在场内，为追求多几个百分点的回报承担损失50%的风险
- **获利后立即放大全部头寸**：应将部分获利存入银行，只用剩余部分扩大规模，否则一次错误即可回吐全部利润

## Key Takeaways
1. 永远先问"我会亏多少"，再问"我能赚多少"——资本保障是所有决策的第一道滤网
2. 任何单一头寸的潜在损失不超过总风险资本的1%~2%，确保永远都有下一次交易机会
3. 严格维持1:3以上的风险/回报比，这让你即使胜率只有33%也能长期获利
4. 亏损时缩小头寸，获利时存入一半再扩大——这是一个自我修正的正反馈系统
5. 当市场不具备合理的风险/回报关系时，持有现金不是懦弱，是纪律
6. "迅速认赔，但让获利头寸持续发展"的精确执行标准是：任何既有获利不可再损失50%
7. 卓越回报不是赌出来的——它是在资本保障和一致性获利被系统化执行后自然浮现的机会
