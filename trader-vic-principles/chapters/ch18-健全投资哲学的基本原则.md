# Chapter 18: 健全投资哲学的基本原则

## Core Idea
斯波朗迪将“哲学”还原为“爱好智慧/知识”的求真态度，并以此构筑交易三原则——资本保障为绝对前提，在此之上自然生出一致性获利能力，待利润累积稳固后才以部分利润追求卓越报酬，三者顺序绝不可逆。

## Frameworks Introduced
- **三原则金字塔**：资本保障 → 一致性获利能力 → 追求卓越报酬。自底向上递进，下层条件不满足则上层不可启动。
  - When to use: 任何交易决策之前，作为资金管理的元规则评估风险状态与可用资本条件。
  - How:
    1. 每笔交易入场前，先确定止损位并计算潜在亏损，确认亏损 ≤ 总资本的 2-3%。斯波朗迪以 3% 为基准。
    2. 以当前可用资本（而非初始资本）的 3% 为每笔交易的风险上限。亏损时风险绝对值自动收缩，盈利时逐步放大。
    3. 每笔交易的风险/回报比必须 ≥ 1:3，即潜在获利至少 3 倍于潜在亏损。不满足此门槛的交易直接放弃。
    4. 当账户由亏损转为盈利时，将每一获利交易的 50% 净利润存入银行，且在该会计期间不再动用这部分资金。
    5. 只有当银行账户中的锁定利润已达初始本金的一定倍数后，方可以部分利润（绝不动用初始本金）进入第三层，参与期权等高赔率交易。
    6. 第三层发生亏损时，退回第二层重新累积利润；第二层出现连续亏损或回撤超阈值时退回第一层，缩减风险资本——绝不向下加码博回本。

## Key Concepts
- **资本保障**：交易第一前提。斯波朗迪认同萨姆·巴斯的法则——“不要发生任何亏损”。以绝对而非相对的视角评估绩效；唯一有效的问题是“我是否赚钱？”，答案为否则缩减风险资本，为是则适度放大。
- **一致性获利能力**：资本保障原则的自然延伸。通过固定比例风险控制与强制利润提取，即便 33.3% 胜率、1:3 赔率，仍可实现年化约 27.08% 收益，最大回撤仅约 5.91%。
- **3% 风险规则**：每笔交易以当前可用资本的 3% 承担风险。斯波朗迪以表 18-1 证明：3 笔连续亏损 + 1 笔 3 倍盈利 = 账户仅亏损 0.5%。即使连续亏损 30 笔，仍保有初始资本的 40.1%。
- **50% 利润锁定**：账户由亏转盈后，每笔盈利的 50% 立即存入银行，当期不再参与交易。核心目的：绝不可将所有获利用来承担风险。这创造了只进不出的资本积累机制。
- **追求卓越报酬**：以更积极的态度处理风险，但仅动用已实现的利润而非初始本金。斯波朗迪个人偏爱期权——可限制下档绝对风险并保留上档可观潜能。他举出自身因差旅错失 OEX 看跌期权 1100% 回报的教训作为反面案例。
- **绝对绩效观**：拒绝“相对收益”的自我欺骗。唯一有效的绩效问题是“我是否在赚钱？”——相对比较（“跑赢指数 5%”但实际亏损）不能弥补绝对资本的减损。
- **风险/回报比 1:3**：贯穿三原则所有层级的最低准入标准。只有当潜在获利至少 3 倍于潜在亏损时才可进场。斯波朗迪指出，结合 3% 规则，每 3 笔交易仅需成功 1 笔即可处于获利状态。

## Mental Models
- **单向不可逆阀门**：三原则金字塔的层级之间不可跨越——资本保障未稳固时一致性获利无从谈起，获利未稳定积累时追求卓越报酬必然消耗本金。任何层级条件恶化时必须退回上一级，而不是向下加码博回撤。
- **概率仓位自动缩放**：3% 规则使仓位规模随可用资本自动浮动——连续亏损时仓位自动缩小（保护残存资本），连续盈利时仓位自动放大（用利润博利润）。这是“赢时筹码重、输时筹码轻”的机制化实现，无需复杂计算。
- **棘轮锁定效应**：50% 利润锁定创造了一个只进不出的资本积累棘轮——利润的一半永久脱离市场风险，如同棘轮只允许齿轮单向转动，从机制层面杜绝“连续盈利后一把亏光”的经典交易悲剧。

## Anti-patterns
- **以相对绩效代替绝对绩效**：声称“虽然亏了 10%，但大盘跌了 20% 所以我表现优秀”——斯波朗迪视此为最严重的自我欺骗。亏损就是亏损，相对比较不能弥补绝对资本损失，且扭曲风险管理能力。
- **全仓复利不锁定利润**：账面浮盈不提取，继续以全部资金承担同等比例风险。一旦遭遇连续亏损，利润加速蒸发。50% 利润锁定正是对此模式的机制化反制。
- **跳过前两层直攻第三层**：在本金尚未稳定获利时使用杠杆或期权博取暴利。违反金字塔的单向依赖——下层不存，上层必塌。
- **凭主观信心突破量化规则**：因“这次胜算很高”而超配仓位、突破 3% 风险上限。一次高信心的失败交易就可能造成远超系统容忍度的回撤。

## Key Takeaways
1. 任何一笔交易入场前，先写下止损位并确认亏损 ≤ 可用资本的 3%，否则放弃交易。
2. 每笔盈利到账后立即执行 50% 利润划转银行账户，使这个动作成为条件反射而非可犹豫的选择。
3. 绩效评估只问一个绝对问题：“我赚钱了吗？”答案为否则降仓位，为是才可维持或适度放大。
4. 只有当银行账户中锁定的利润已达到初始本金的数倍后，才考虑以利润中的一小部分参与高风险高赔率交易。
5. 三原则的先后排序是硬约束——保障资本出现问题（连续亏损、回撤超阈值）时，必须从高层退回，而非加码博回本。
6. 将 1:3 风险/回报比设为所有交易的铁门槛：不满足此赔率的机会根本不应进入下一步分析。
