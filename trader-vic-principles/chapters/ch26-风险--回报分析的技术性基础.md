# Chapter 26: 风险——回报分析的技术性基础

## Core Idea
技术分析的真正价值不在于预测，而在于为每一笔交易提供可量化、可验证的风险参数（止损位）与回报参数（目标位），使交易者能够客观计算风险-回报比，从而只参与期望值为正的交易。

## Frameworks Introduced
- **技术性风险-回报分析法**：风险=入场价−技术性止损价，回报=目标价−入场价，比值≥1:3方可交易
  - When to use: 每次开仓决策前，无论依据何种信号
  - How:
    1. 确认当前趋势方向（道氏理论判定主要/次级/小型趋势）
    2. 识别最近的显著技术位：支撑（做多）或阻力（做空）
    3. 等待1-2-3准则或2B准则在该技术位附近触发入场信号
    4. 止损设在信号失效点：1-2-3模式中设在点2外侧，2B模式中设在点1外侧
    5. 衡量目标价位：使用形态高度投影法或下一显著技术位
    6. 计算 R/R = (目标−入场) / (入场−止损)，仅当≥1:3时执行

- **形态高度投影法（Measured Move）**：目标价=突破点±形态高度
  - When to use: 当价格突破明确的整理形态（矩形、头肩、三角形）或1-2-3/2B结构时
  - How:
    1. 测量形态的垂直高度 H（从最高点到最低点）
    2. 突破方向为向上时，目标=突破价+H
    3. 突破方向为向下时，目标=突破价−H
    4. 若目标与下一技术位重叠，以较保守者为准

- **技术锚定止损法**：止损永远设在技术面失效点，而非固定金额或百分比
  - When to use: 每次设定止损时
  - How:
    1. 问自己："价格运动到哪个点位，我的交易理由就不再成立？"
    2. 将该点位确认为止损位（做多时略低于支撑，做空时略高于阻力）
    3. 用该止损位反推仓位规模（资金管理公式：仓位=可承受亏损/每单位风险）
    4. 若反推出的R/R<1:3，放弃交易——不要移动止损来凑比值

## Key Concepts
- **技术性止损**：基于图表结构失效点（如支撑跌破、趋势线破位、形态高点被穿越）设置的止损，而非主观金额止损
- **目标价位**：通过形态测量或下一技术位得出的潜在盈利终点，用于计算回报端
- **风险-回报比（R/R Ratio）**：潜在亏损与潜在盈利的比值，斯波朗迪要求最低1:3
- **期望值（Expected Value）**：EV = 胜率×平均盈利−(1−胜率)×平均亏损，高R/R配合合理胜率才能产生正期望
- **信号失效点**：价格穿越该点即意味着入场依据被市场否定——这是唯一合理的止损位置
- **资金管理联动**：止损位决定每单位风险，进而决定仓位规模；R/R决定是否交易，二者不可分割
- **技术位优先级**：长期趋势线>中期趋势线>显著高低点>均线>短期密集区，优先使用高级别技术位作为参数锚定
- **多重时间框架验证**：在更高级别时间框架确认技术位有效性后，再在交易级别执行入场

## Mental Models
- **交易理由必须可证伪**：每一笔交易必须有一个明确的技术点位，价格到达该点位即证明你的判断错误。如果没有这样的点位，说明交易逻辑不清晰，不应入场
- **先定风险，后算回报，再决取舍**：止损位必须独立于目标位先行确定。先确定"错在哪里"，再估计"对到哪里"，最后计算比值决定是否交易。不可先看到大目标再回头勉强凑止损
- **概率思维与赔率思维的统一**：R/R是赔率，胜率是概率，二者相乘才是期望值。高赔率不代表可以忽略胜率——斯波朗迪要求技术信号本身必须经过历史验证（如1-2-3和2B的统计有效性）
- **交易是一系列下注，不是一次赌博**：单笔交易的R/R即使达到1:5也可能亏损。真正的优势在于，当你在100笔交易中严格执行1:3以上R/R时，即使胜率仅35-40%，长期期望值也为正

## Anti-patterns
- **固定金额/百分比止损**：不参照技术位，仅凭"每笔亏500美元"或"亏2%就砍"设定止损，导致止损频繁被噪声触发或被做市商猎杀
- **倒推凑比值**：先有了"想做多"的结论，再在图表上寻找一个遥远的阻力位充当目标，强行把R/R算到1:3以上。目标必须是客观技术位，不能是愿望
- **忽略趋势级别的R/R**：在主要趋势向下时，因为日线出现一个看似高R/R的做多信号就入场。逆主趋势的高R/R往往是陷阱——因为胜率被趋势方向严重压缩
- **止损过紧（噪音止损）**：将止损设在仅比入场价差几个跳动点的位置以追求纸面上的高R/R，但该止损没有技术意义，极易被正常波动扫出

## Key Takeaways
1. 所有交易参数（入场、止损、目标）必须锚定在客观的技术价位上，不可凭空设定
2. 止损位先于目标位确定——"我错在哪里"比"我赚多少"更基础
3. 风险-回报比低于1:3的交易直接放弃，不妥协，不找借口
4. 仓位规模由技术性止损位反推：愿意承受的总亏损÷每单位技术风险=可开仓位
5. 高R/R必须与合理的胜率结合，胜率来自经过统计验证的技术信号（1-2-3、2B等）
6. 趋势是你的第一过滤器——顺趋势方向的高R/R信号才具有真实优势
