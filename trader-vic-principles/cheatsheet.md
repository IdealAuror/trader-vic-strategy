# 专业投机原理 决策速查表 — Victor Sperandeo

## 一、决策规则：When X → Do Y, Because Z

| 条件 X | 行动 Y | 理由 Z |
|---|---|---|
| 价格收盘穿越正确绘制的趋势线 | 标记趋势变动第一征兆，准备行动 | 趋势线突破是反转信号序列的起点 |
| 上升趋势价格不创新高 / 下降趋势不创新低 | 激进者轻仓试探，保守者等步骤3 | 动能衰竭客观证据 |
| 价格跌破前回档低点（上升）或升破前反弹高点（下降） | 建仓至完整头寸 | 1-2-3三步全满足=趋势已变 |
| 价格创新高后迅速回落跌破前高（上升2B） | 以止损单做空，止损设于假突破高点之上 | 清除止损的流动性陷阱，极小止损博极大空间 |
| 价格创新低后迅速回升突破前低（下降2B） | 以止损单做多，止损设于假突破低点之下 | 同上，风险回报不对称窗口 |
| BC浪成交量萎缩，价格突破BC趋势线 | 在C点沿主趋势方向建仓 | 次级修正结束，回归主趋势 |
| 窄幅盘整2-3周+（约5%区间），价格放量突破区间上限 | 做多 | 承接完成，机构资金进场 |
| 窄幅盘整后价格放量跌破区间下限 | 做空 | 出货完成，机构离场 |
| 趋势线上方/下方出现跳空缺口且穿越趋势线 | 沿缺口方向建仓，不需等2B确认 | 缺口反映重大基本面突变 |
| 中期走势出现连续四天朝一方向运行后反转（四天准则） | 反向建仓；第1天出现逆趋势行情即可入场 | 25%立即反转，41%在6天内反转，75%在24天内反转 |
| 开盘后10-15分钟缺口未被填补 | 沿缺口方向交易全天 | 95%缺口在10-15分钟内填补；未填补=方向确认 |
| 价格突破开盘区间（开盘前几分钟高低点）上限/下限 | 沿突破方向建仓 | 开盘区间决定全天主要趋势方向 |
| 连续下跌3天后买进（多头中期上升趋势中） | 第3天收盘买进，持有1-2天获利了结 | 历史统计：胜率94.4%，平均涨幅1.309% |
| 风险回报比不足1:3 | 不交易，持有现金 | 正期望值是长期获利的数学基础 |
| 价格达中期目标 | 平仓50%+收回本金，止损推至保本，剩余仓位追踪 | 保障资本→一致性获利→卓越回报递进 |
| 风险回报比从1:3降至1:2 | 止损调至成本价之上 | 头寸从此"只可能获利不可能亏损" |
| 任何头寸出现亏损且判断错误 | 立即无条件平仓，不可加仓/对冲/调仓 | 鳄鱼原则——越挣扎损失越大 |
| 报价滞后（快速市场） | 绝对不建仓 | 荧屏报价不可靠 |
| 经济消息公布后市场反应与消息方向相反 | 视为趋势反转辅助确认 | 利好不涨=利空，利空不跌=利好 |
| "应该发生的事情没有发生"（蓝图预测法） | 立即反转头寸 | 市场用价格投票否定你的假设 |

## 二、入场/离场决策树

```
趋势方向判断（道氏理论 + 双指数相互确认）
│
├─ 顺势方向明确 ──────────────────────────────────
│  ├─ 1-2-3步骤3完成 → 完整仓位进场
│  ├─ 2B触发（中3-5天/长7-10天窗口最佳）→ 轻仓先锋进场
│  │  └─ 1-2-3步骤3确认 → 加仓至完整 + 止损统一收束
│  ├─ ABC三浪C点 + BC量缩+突破BC趋势线 → 投机性仓位
│  ├─ 四天准则触发 → 反向建仓（中期顶底专用）
│  └─ 缺口穿越趋势线 + 另一准则确认 → 沿缺口方向进场
│
├─ 横盘整理（窄幅盘整2-3周+，约5%区间）
│  ├─ 放量突破区间上限 → 做多
│  └─ 放量跌破区间下限 → 做空
│
├─ 当日冲销专用 ─────────────────────────────────
│  ├─ 开盘区间突破 → 沿突破方向交易
│  ├─ 跳空后10-15分未被填补 → 缺口方向确认全天
│  └─ 七项评估系统（收盘后评分）→ 7/7多头→次日90%高价上涨
│
└─ 趋势不明或风险回报<1:3 → 持有现金，等待

离场触发
├─ 价格触及技术止损位 → 立即平仓
├─ 价格达第一目标位 → 平50% + 止损推至保本 + 剩余追踪
├─ 时间止损（5-10根K线未朝预期方向移动）→ 离场
├─ 心存怀疑时 → 立即出场
├─ "挨耳光"后出现反向信号 → 果断反向建仓
└─ 亏损头寸不过夜（当日冲销铁律）
```

## 三、信号可靠度矩阵

| 信号类型 | 可靠性 | 适用周期 | 核心要求 |
|---|---|---|---|
| 1-2-3三步全满足 | 最高 | 短/中/长 | 趋势线必须精确绘制 |
| 2B（中3-5天/长7-10天） | 高（胜率约50%，盈亏比≥1:5） | 短/中/长 | 假突破缩量+反转放量 |
| ABC三浪C点突破 | 高（顺势） | 中/长 | BC量缩+价格突破BC趋势线 |
| 四天准则（中期专杀） | 高（75%在24天内反转） | 中期 | 连续4天单向运行后出现逆向首日 |
| 缺口穿越趋势线 | 中高 | 任意 | 必须穿越趋势线+另有一准则确认 |
| 开盘区间突破 | 中高（当日） | 日内 | 开盘区间定锚+反方向止损 |
| 三天高低价准则 | 中 | 短期 | 必须配合确认原则 |
| 七项评估系统（7/7） | 中（90%次日高价上涨） | 次日 | 全7项得分稀有但可靠 |
| 摆荡指标背离 | 中（辅助过滤） | 短/中 | 仅过滤，非独立信号 |
| 单一步骤2（无步骤3） | 低——不可单独使用 | 任意 | 必须配合2B轻仓试单 |
| 趋势线随意绘制后突破 | 无效 | — | 锚点间穿越价格=无效趋势线 |
| 200天均线交叉 | 低（严重滞后） | 长 | 崩盘中几乎无用，仅作背景参考 |

## 四、阈值与公式

| 参数 | 阈值 | 公式/说明 |
|---|---|---|
| 单笔风险上限 | 总风险资本2%（保守）-5%（激进上限） | 先定技术止损位，反推仓位 |
| 风险回报比下限 | 1:3（铁律，不可妥协） | 回报=入场-目标，风险=入场-止损 |
| 仓位计算 | 仓位 = 可承受亏损金额 ÷ |入场价-止损价| | 止损距离由1-2-3/2B信号决定 |
| 获利保护 | 任何既有获利不可再损失>50% | 锁定已实现利润的具体标准 |
| 获胜保护升级 | 风险回报比从1:3→1:2时止损移至保本 | 头寸从此零风险 |
| 胜率与盈亏比方程 | 胜率≥33%即可（1:3支撑） | 期望值=0.33×3-0.67×1=+0.32R |
| 趋势寿命判定 | 运行至历史均值50%分位时提高警惕 | 基于1896至今正态分布统计 |
| 2B确认窗口 | 短期≤1天、中期3-5天、长期7-10天 | 停留过短不可靠 |
| 道氏次级修正范围 | 幅度33%-66%，持续3周-3个月 | 超出此范围警惕趋势反转 |
| 多头市场均值 | 2.33年 / 涨幅77.5% | 1896-1985道琼斯统计 |
| 空头市场均值 | 1.1年 / 跌幅29.4% | 同上 |
| 股市领先经济 | 3-6个月（底部）/ 6-9个月（顶部） | 非精确物理定律，需技术确认 |
| 期权单笔上限 | 风险资本2%-3%，极端确信≤10% | 期权盈亏比要求≥1:5 |
| 当日冲销四阶段 | ①开盘反走0.5h ②10:00-12:00单向 ③12:00-14:30反向 ④15:10后80%概率收于15:10方向 | S&P期货典型日内节奏 |

## 五、资金管理三级系统

| 层级 | 内容 | 工具 |
|---|---|---|
| 第一层 | 风险回报比评估 | 技术分析确定入场/止损/目标，计算比值 |
| 第二层 | 成功几率评估 | 历史正态分布（幅度+期限）+ 技术面+基本面交叉 |
| 第三层 | 资产动态配置 | 仓位=0%-100%浮动，非二元满仓空仓 |
| 配置比例公式 | 仓位=最高配置比×存活率 | 例：60%×69%=41.4%（基于103天/12%回撤数据） |

市场风险评估八问：长期趋势方向→长期趋势阶段→中期趋势历史对比→道氏信号（背离/量/宽度）→均线→摆荡指标→基本面六项（通胀利率/负债信用/利率债券/美元汇率/消费者心态/产业领导股/潜在利空）→是否存在重大谬论可利用

## 六、情绪预警 & "臭味"检测

| 情绪信号 | 臭味检测问题 | 对应七特质缺陷 |
|---|---|---|
| "这一次例外" | 我有客观证据支持例外，还是在满足愿望？ | 诚实、客观 |
| 亏损后感到"不公平" | 我的什么行为导致了这一结果？ | 诚实 |
| "脑子说停，心里想做"（史波克症候群） | 我的情绪在告诉我什么价值观冲突？ | 情绪控制 |
| 拒绝止损，期待"一定回来" | 我根据现实还是希望做决策？ | 勇气、果断 |
| 因"已经是历史低价"买入 | 趋势变动信号是什么？价格不是理由 | 客观 |
| 获利后立即想平仓 | 恐惧来自客观信号还是逃避不确定性？ | 自律、耐性 |
| 错过最佳点位后拒绝顺势 | 我是不愿"完美正确"还是宁可"错误错过"？ | 勇气 |
| 亏损后加大仓位 | 这是新开仓我会在这里入场吗？ | 自律（摊平=最致命错误） |
| 炫耀胜率但账户净亏 | 我的净盈亏数字是多少？ | 诚实 |
| 寻求他人意见后责怪那人 | 这笔交易决策是我做的吗？ | 诚实（责任不可外包） |
| "我需要这笔交易赢" | 这是"我希望"还是"我必须"？后者毁灭 | 情绪控制 |
| 因紧张兴奋失去睡眠 | 潜在亏损是否超过总资本2%？ | 自律（仓位过大） |
| 长期不看盘/逃离办公室 | 这是情绪逃避还是意识选择？ | 勇气、果断 |
| 连续亏损后报复交易 | 我现在是在执行系统还是在宣泄情绪？ | 耐性、自律 |
| 盈利后立即放大仓位 | 仓位与账户规模挂钩，不与短期业绩挂钩 | 自律 |

## 七、亏损诊断四象限（ch12）

| | 知识不足 | 情绪驱动 |
|---|---|---|
| **执行失败** | 无知型：缺乏市场知识就进场 → 补学再交易 | 冲动型：盘中临时改主意 → 设15分钟冷却期 |
| **系统缺陷** | 系统型：方法本身有漏洞 → 修正或更换系统 | 纪律型：明知规则却未遵守 → 强化清单核查 |

## 八、成功交易者七特质自检（ch31）

自律→耐性→勇气→客观→果断→情绪控制→诚实。七者连锁，缺一会导致其他失效。每日交易前逐项打分（1-5），任一项<3不开仓。心理占成功80%，技术仅20%。

## 九、核心心理复原程序

1. 接受情绪——不搏斗，暂停冲动反应
2. 辨识根源——这个情绪对应潜意识中的什么价值观？
3. 评估有效性——当前事件背景是否合理？（祖先面对野猪=有效，交易面对K线=过时）
4. 设想后果——按情绪行动，5分钟/5天/5年后怎样？
5. 选择行动——你现在有选择的余地

## 十、三重原则递进（不可颠倒）

**保障资本**（先问"我会亏多少"）→ **一致性获利**（亏损缩小头寸，获利一半存入一半扩大）→ **卓越回报**（仅以累积利润博更高风险）
